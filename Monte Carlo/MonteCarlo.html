Overview

Monte Carlo simulations are useful for understanding estimator properties (e.g., unbiasedness, consistency). In general, they consist of the following steps.

  • Step 1: assume a known data generating process (DGP), e.g., \(y_{i}=\beta X_{i}+e_{i}\)
  • Step 2: generate a random sample of size \(n\) from the DGP.
  • Step 3: compute desired statistics (e.g., mean, sd, beta) and save them.
  • Step 4: redo steps 2-3 many times.
  • Step 5: look at the distribution of saved statistics across iterations.

In the following, we will learn how to run Monte Carlo simulations using R with an example of omitted variable bias.

Concept Review

Consider the following DGP: \[y_{i}=\beta_{1}x_{1i}+\beta_{2}x_{2i} +\epsilon_{i}, \] where \(E[x_i\epsilon_i]=0.\) Suppose you estimate the model using OLS but omitting \(x_{2i}\): \[y_{i}=\alpha_{1}x_{1i}+e_{i} \]

Then, \[ \begin{align} \alpha_{1}&=\frac{cov(x_{1i},y_i)}{var(x_{1i})}\\ &=\beta_1+\beta_2\frac{cov(x_{1i},x_{2i})}{var(x_{1i})}+ \overbrace{\frac{cov(x_{1i},\epsilon_{i})}{var(x_{1i})}}^{=0}\\ &=\beta_{1}+\beta_2 \gamma,\\ \end{align} \]

where \(\gamma\) is from \(x_{2i}=\gamma x_{1i}+\eta_{i}.\)

As the equation suggests, omitted variable bias arises if

  • \(\beta_{2}\neq 0\): the omitted variable is correlated with the dependent variable AND
  • \(\gamma\neq 0\): the omitted variable is correlated with your right hand side variable in your regression.

pmap: Apply a list of inputs (multiple dimensions) to a function

bind <- function(input1,input2){paste0(input1,input2)}

You can return multiple outputs in each iteration and combine the results in a data frame.

bind <- function(input1, input2) {
  list(
    output1 = paste0(input1, input2),
    output2 = paste0(input2, input1)
  )
}

pmap(list(param$x, param$y), bind) %>%
  bind_rows()

If the names of inputs and functions are identical, you can simplify your code.

bind <- function(x, y) {
  list(
    output1 = paste0(x, x),
    output2 = paste0(y, x)
  )
}

pmap(param, bind) %>%
  bind_rows()

Coding

We will generate data of the form: \[y_{i}=\beta_{1}x_{1i}+\beta_{2}x_{2i} +\epsilon_{i},\] where \(x_{2i}=\gamma x_{1i}+\eta_{i}\), \(x_{1i} \sim N(0,1)\) and \(\eta_{i} \sim N(\sqrt{1-{\gamma}^2},1)\) for \(i\in\{1,2,...,100\}\).

For each simulation, we will estimate \(\alpha_1\) from \[y_{i}=\alpha_{1}x_{1i} +e_{i},\]

We will consider all the following possible combinations of DGP:

  • \(\epsilon_{i}: N(0,1)\) and student-\(t\) distribution with \(df=5\),
  • \(\gamma \in\{0,.5,1\}\),
  • \(\beta_{2} \in\{0,.5,1\}\)

For each combination of DGP (2 x 3 x 3 = 12), we will do 200 simulations.

Let’s start by setting things up.

#--- load packages ---#
library(tidyverse)
library(purrr)

#--- set seed ---#
set.seed(2021) 

#--- number of repetitions ---# 
num_reps <- 200

We first create a function that 1) randomly draw \(n\) samples from DGP and 2) estimate \(\alpha_1\).

sim_func <- function(rep, n, gamma, b1, b2, dist_sim) {
  #--- DGP ---#
  df <- tibble(
    x1 = rnorm(n = n, 0, 1),
    x2 = gamma * x1 + sqrt(1 - gamma^2) * rnorm(n = n, 0, 1),
    err = dist_list[[dist_sim]](n),
    Y = b1 * x1 + b2 * x2 + err
  )
  
  #--- estimate alpha ---#
  reg <- lm(Y ~ 0+x1, data = df) # ignore intercept
  
  #--- store results of interest (potentially many) ---#
  results <- list(alpha = coefficients(reg)[1]) # put results in a list
  return(results)
}
We then specify all possible combinations of parameters. 
param_list <- expand.grid(
  rep = 1:num_reps,
  n = 100, 
  gamma = c(0, 0.5, 0.9),
  b1 = 1, 
  b2 = c(0, 0.5, 1), 
  dist_sim = c("Normal", "Student_t")
)

dim(param_list)
We then specify what Normal and Student_t means. 
dist_list <- list(
  Normal = function(n) rnorm(n, 0, 1),
  Student_t = function(n) rt(n, 5)
)
Before we generalize, let’s run a few. 
param_123 <- param_list[1:3,]
pmap(param_123, sim_func) %>%
  bind_rows()

It looks good. Now, let’s combine our parameters and results.

sim_out <-
  param_list %>%
  mutate(results = pmap(param_list, sim_func)) %>%
  unnest_wider(results) %>%  # this is useful if we stored multiple outputs
  group_by(gamma, b2) %>%
  mutate(mean_alpha = mean(alpha))

sim_out %>% head()
